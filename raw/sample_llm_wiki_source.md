# Sample LLM Wiki Source

Date: 2026-04-26

## Purpose

This sample source validates the Karpathy LLM Wiki workflow.

## Key Ideas

- Raw sources stay immutable.
- The LLM compiles durable knowledge into `wiki/`.
- `wiki/index.md` helps the LLM navigate at query time.
- `wiki/log.md` records the evolution of the knowledge base.
- Good answers can become durable analysis pages.

## Expected Outputs

- `wiki/sources/Sample_Source.md`
- `wiki/concepts/LLM_Wiki_Pattern.md`
