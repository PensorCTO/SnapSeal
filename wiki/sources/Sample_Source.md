# Sample Source

## Core Synthesis

This source establishes the baseline operating model for the LLM Wiki. It introduces immutable raw sources, LLM-owned compiled wiki pages, index-first querying, chronological logging, and the practice of saving durable analysis back into the wiki.

## Key Claims

- Raw source files should remain immutable.
- The LLM should compile knowledge into `wiki/`.
- `wiki/index.md` should guide query-time navigation.
- `wiki/log.md` should record important activity.
- Durable answers can become pages in `wiki/analyses/`.
